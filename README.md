# 🌱 Carbon Footprint Calculator

## 📌 Project Overview

The Carbon Footprint Calculator is a full-stack web application that estimates the carbon emissions generated from e-commerce shipping activities. The system helps users understand the environmental impact of their purchases and encourages sustainable decision-making by providing actionable insights and carbon reduction suggestions.

This project demonstrates the integration of modern web technologies to solve real-world environmental challenges.

---

## 🎯 Objective

* To calculate CO₂ emissions based on shipping parameters
* To promote environmental awareness among users
* To provide eco-friendly alternatives and recommendations
* To build a scalable and user-friendly web application

---

## 🛠️ Technologies Used

| Layer    | Technology           |
| -------- | -------------------- |
| Frontend | React + Tailwind CSS |
| Backend  | Node.js + Express    |
| Database | MongoDB (Mongoose)   |
| Auth     | JWT + bcrypt         |

---

## 🧩 System Architecture

The application follows a **client-server architecture**:

* **Frontend (React):**

  * Handles user interaction and UI rendering
  * Collects input data such as distance, weight, and transport mode

* **Backend (Node.js + Express):**

  * Processes user requests
  * Implements business logic for carbon calculation
  * Handles authentication and API endpoints

* **Database (MongoDB):**

  * Stores user data and calculation history
  * Ensures efficient data retrieval and scalability

---

## ⚙️ How It Works

1. User enters shipping details (distance, package weight, transport type)
2. The backend calculates carbon emissions using predefined emission factors
3. The result is sent back to the frontend
4. The UI displays:

   * Total CO₂ emissions
   * Environmental impact insights
   * Suggestions to reduce carbon footprint

---

## 🔐 Authentication Flow

* User registration and login are handled securely
* Passwords are hashed using bcrypt
* JWT (JSON Web Token) is used for session management
* Protected routes ensure authorized access
